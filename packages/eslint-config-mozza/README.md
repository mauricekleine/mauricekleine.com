## About the project

This package contains custom ESLint rules based on:

- [eslint-config-next](https://github.com/vercel/next.js/tree/canary/packages/eslint-config-next)
- [eslint-config-prettier](https://github.com/prettier/eslint-config-prettier)
- [eslint-config-turbo](https://github.com/vercel/turbo/tree/main/packages/eslint-config-turbo)
- [eslint-plugin-import](https://github.com/import-js/eslint-plugin-import)
- [eslint-plugin-sort](https://github.com/mskelton/eslint-plugin-sort)
- [eslint-plugin-testing-library](https://github.com/testing-library/eslint-plugin-testing-library)
- [eslint-plugin-unicorn](https://github.com/sindresorhus/eslint-plugin-unicorn)
- [eslint-plugin-unused-imports](https://github.com/sweepline/eslint-plugin-unused-imports)
